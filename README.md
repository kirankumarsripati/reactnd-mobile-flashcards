# React Native Mobile Flashcards

This is the final project in the React Nanodegree Program,
to evaluate the learnings of React Native.

This app allows users to study collections of flash cards. Create
decks and add flash cards to deck. Use quiz to test yourself.

## Installation
1. Install all dependencies with `yarn install`
2. Start the development with `yarn ios` or `yarn android` or use `yarn web` to
to view in the browser.
